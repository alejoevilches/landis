# Landis
Landis is a store made using React and consuming the FakeStore API.

## Description

It was made using JavaScript and [React.js](https://reactjs.org/). The styles were made using pure CSS

## Features

- Search products
- Filter products by category
- Add products to a wish list
- Add products to the cart
- Delete products from the wish list
- Delete products from cart

## Requirements

- [Node.js](https://nodejs.org/) installed.
- NPM (it comes with Node.js).

## Install

1. Clone this repository in your computer:

   ```bash
   git clone https://github.com/alejoevilches/landis.git

2. Navigate to the project directory
    ```bash
    cd landis

3. Install dependencies

    ```bash
    npm install 

4. Run the development server:
    ```bash
    npm run dev
5. Access your localhost route to test the app.

## Deploy
If you want to test the app without installing it, you can check the deploy [here](https://landis.netlify.app/)
